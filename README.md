### Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/carmineroberto14-creator/progress-hub.git
   ```
2. Navigate to the project directory:
   ```
   cd progress-hub
   ```
3. Install the required dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```
